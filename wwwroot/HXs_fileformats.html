Tubesheet File Format

The name of the file should be the name of the HX to be added to the program

Example: Example HX.csv
    
        x_label;y_label;cl_x;cl_y;hl_x;hl_y;tube_id
        2;1;71,938;0,704;0,812;0,704;TUBE.1
        4;1;71,125;0,704;1,625;0,704;TUBE.2
        6;1;70,312;0,704;2,438;0,704;TUBE.3
        8;1;69,5;0,704;3,25;0,704;TUBE.4

HX config file:

The name of the file should be: config.json

Example: config.json
    
    {
        "leg" : "cold",                 // "hot", "cold" or "both"
        "tube_od" : 12.900000006,       // tube outside diameter
        "min_x" : -304.8,               // minimum x coordinate of the viewport for the tubesheet
        "width" : 609.6,                // width of the viewport for the tubesheet
        "min_y" : -304.8,               // minimum y coordinate of the viewport for the tubesheet
        "height" : 609.6,               // height of the viewport for the tubesheet
        "font_size" : "5.08",           // font size for coordinates
        "x_labels" : "-297.18 297.18",  // where to put coordinates on the horizontal axis, many can be used, separate them by space
        "y_labels" : "-292.1 292.1",    // where to put coordinates on the vertical axis, many can be used, separate them by space
        "unit" : "mm"                   // "mm" or "inch"
    }

Important: Both files should be supplied simultaneously
